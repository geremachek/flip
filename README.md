<p align="center">A simple command that lets you "flip" through files</p>
<br><br>

**Usage:**

```
flip [FILE...]
```

**Keybinds:**

<kbd>h</kbd> / <kbd>LEFT</kbd>: Move back one file

<kbd>l</kbd> / <kbd>RIGHT</kbd>: Move forward one file

<kbd>q</kbd>: Quit

<kbd>b</kbd>: Toggle the status bar
